### Installation
1. Clone the repo
   ```sh
   git clone git@github.com:BubbleKID/blizzard.com-clone-client.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run dev server
   ```sh
   npm run dev
   ```
